JavaScript

JavaScript는 웹 페이지를 동적으로 만들고 조작하기 위해 사용되는 프로그래밍 언어이다.
? 웹 페이지에서 발생하는 이벤트(버튼클릭, 마우스 오버, 키보드 입력 등)에 반응하는 신경계와 같은 역할을 한다고 볼 수있다. 이런 특정 동작을 수행함으로써 웹 페이지의 내용을 동적으로 제어할 수 있다.

아래 슬라이드는 JavaScript를 활용한 다양한 예제들이다. 응용가능한 G SAP, LOCOMOTIVE SCROLL, SCROLLA등 다양한 라이브러리들을 활용하여 재미있는 디자인을 만들 수 있다.
